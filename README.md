# json2dart_serialization
一个网页版本的 json 转 dart 库
## 说明
json_serializable 可以参考 flutter 官网 json 部分 [flutter](https://flutter.io/json)
项目使用 dart web 开发
## 开发环境
- 本地启动
```shell
 dart pub global activate webdev
 // 运行
  webdev serve
```
